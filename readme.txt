Макет явно предполагает полноэкранный формат, поэтому некоторые
размеры не соответствуют макетным, подгонял с учетом размеров появляющего текста по AJAX запросу.
Выполнены: анимация, подчеркивание активной страницы,
AJAX запрос на смену текста, длина текста ограничена по количеству
строк для не высоких экранов, в поле ввода реализован запрет на ввод определенных
символов, а так же выставлено ограничение на min и max количество вводимых символов.
Выполнен адаптив под различные (не все) варианты экранов. Реализовано меню Бургер
